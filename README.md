# 🎯 Face Recognition Based Attendance System

An AI-powered attendance system that uses **face recognition technology** to automatically mark attendance in real-time. This project eliminates the need for manual attendance and enhances accuracy, security, and efficiency.

---

## 📖 Description

The **Face Recognition Based Attendance System** is designed to automate the traditional attendance process using computer vision and machine learning techniques. The system captures facial data through a webcam, detects and recognizes faces, and marks attendance accordingly.

It leverages advanced image processing algorithms to identify individuals with high accuracy and stores attendance records digitally. This project demonstrates the practical implementation of **AI in real-world applications**, reducing human effort and minimizing errors.

The system can be used in **schools, colleges, offices, and organizations** for seamless and contactless attendance tracking.

---

## 🚀 Features

* 👤 Real-time face detection and recognition
* 📸 Automatic attendance marking
* 🧠 High accuracy using trained ML models
* 🗂️ Attendance stored in CSV/database
* ⏱️ Time-efficient and contactless system
* 💻 Easy to use interface

## ⚙️ How It Works

1. Capture facial images and create dataset
2. Train the model using stored images
3. Detect faces through webcam in real-time
4. Recognize faces using trained model
5. Mark attendance automatically with date & time
